# Poland Biodiversity Dashboard
Using the biodiversity data, built a Shiny dashboard that visualizes observed species on the map.

## Running the app
- Required libraries: 
library(shiny)
library(shinydashboard)
library(tidyverse)
library(lubridate)
library(leaflet)
library(plotly)
library(scales)
library(sf)
library(janitor)
